Files Structure

          assets/css (Template CSS)
          assets/images (All Images)
          assets/js (Template JS)

This documentation is to help you regarding each step of customization. Please go through the documentation carefully to understand how this template is made and how to edit this properly. Basic HTML and CSS knowledge is required to customize this template. 

Requirement

  • Text editor (VSCode, sublime, atom e.t.c)
  • Web browser (Morzilla Firefox, Google Chrome e.t.c)

    • HTML Structure

    This project consist of 4 html pages namely:

              index.html (Homepage)
          contact.html (Contact page)
          portfolio-one.html (Portfolio-one page)
          portfolio-two.html (Portfolio-two page)

    CSS Structure

    The organization of the CSS is one of our priorities.
    These are the CSS file we're using in the template:
      
    - assets/
  - css/
    - style.css                 - (Main CSS file) 
    - home.css                  - (HomePage Specific CSS file) 
    - header.css                - (consistent page header CSS file) 
    - footer.css                - (consistent footer CSS file) 
    - contact.css               - (Contact Specific CSS file) 
    - responsive.css            - (Responsive Styling CSS file) 
    - aos.css                   - (Aos animation CSS file)  

    - bootstrap/bootstrap.min.css - (Bootstrap Minified CSS file)

  - fonts/  - contains fonts for Poppins (Google fonts)

    JavaScript

    jQuery - is a Javascript library that greatly reduces the amount of code that you must write.
    For more information, please visit http://www.jquery.com/
     
    All the JavaScript libraries and files are included at the bottom of every HTML page like this: 
    <!-- Main Jquery JS -->
<script src="./assets/js/jquery/jquery-3.5.1.min.js" type="text/javascript"></script>        
<!-- Popper JS -->
<script src="./assets/js/popper/popper.min.js" type="text/javascript"></script> 
<!-- Bootstrap JS -->
<script src="./assets/js/bootstrap/bootstrap.min.js" type="text/javascript"></script>       
<!-- Carousel JS -->
<script src="./assets/js/carousel.js" type="text/javascript"></script>    
<!-- aos JS -->
<script src="./assets/js/aos.js" type="text/javascript"></script>   
<!-- Custom JS -->
<script src="./assets/js/script.js"></script>

     

    Fonts

    To change the embedded font, please take a look in the head part of assets/css/style.css:
      
                @import url('https://fonts.googleapis.com/css2?family=Poppins:wght@100;200;300;400;500;600;700;800&display=swap');
